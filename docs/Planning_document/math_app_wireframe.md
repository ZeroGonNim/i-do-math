# 📱 4학년 수학 학습 앱 와이어프레임 기획서

## 🎯 기획 개요
* **타겟 사용자:** 10세 (초등학교 4학년)
* **핵심 기능:** 인터랙티브 애니메이션, 단계별 풀이 시각화, 학습 기록 유지, 스마트 오답 리마인드
* **디자인 방향성:** 직관적인 UI, 애니메이션을 통한 즉각적인 피드백, 텍스트 최소화 및 시각적 이해도 향상

---

## 🗺️ 전체 화면 흐름도 (Screen Flow)
1. **[ 홈 화면 ]** 👉 단원 선택 및 진도 확인
2. **[ 문제 풀이 화면 ]** 👉 인터랙티브 애니메이션 조작 및 정답 입력
3. **[ 풀이 결과 화면 ]** 👉 정답/오답 확인 및 단계별 풀이식 시각화
4. **[ 수학 일기 (학습 기록) ]** 👉 과거 풀이 궤적 및 오답 데이터 확인
5. **[ 스마트 리마인드 ]** 👉 유사 오답 발생 시 과거 기록 비교 및 힌트 제공

## 🖥️ 상세 화면 설계 (Wireframe)

### 1. 홈 화면 (Home View)
> **[ 상단 영역 : GNB ]**
> 👤 프로필 (Lv.3 수학 탐험가) | ⭐️ 획득한 별: 150개
> 
> **[ 중앙 영역 : 단원 선택 슬라이더 ]**
> ◀️ [ 1학기 2단원: 각도 ] ▶️
> * 상태바: 현재 단원 달성률 60% (게이지 애니메이션)
> * 대표 이미지: 각도기를 들고 있는 캐릭터
>
> **[ 하단 영역 : CTA 버튼 ]**
> 🟢 [ 이어서 학습하기 ]

### 2. 문제 풀이 화면 (Problem Solving View)
> **[ 상단 바 : Header ]**
> ⬅️ 나가기 | 진행 상황: 🟢🟢🟢⚪⚪ (3/5) | 💡 힌트 보기
> 
> **[ 🎬 시각화 & 애니메이션 캔버스 (화면의 40%) ]**
> * **Default:** 문제에 맞는 도형, 분수 조각 등 렌더링
> * **Interaction:** 사용자가 직접 각도기를 회전시키거나 도형을 드래그하는 영역
> 
> **[ 📝 문제 텍스트 영역 ]**
> Q. 피자 한 판을 8조각으로 나누었습니다. 그 중 3조각을 먹고, 2조각을 더 먹었다면 먹은 피자는 전체의 얼마인가요?
> 
> **[ ⌨️ 커스텀 입출력 영역 (Bottom Sheet) ]**
> * **입력창:** ( 빈칸 ) / ( 빈칸 )
> * **커스텀 키패드:** 1~9 숫자, 지우기, 분수 기호(/) 등 큼직한 버튼
> * **CTA 버튼:** 🔵 [ 정답 확인 ] (입력 시 활성화)

### 3. 풀이 결과 및 단계별 해설 화면 (Solution & Feedback View)
> **[ 상단 팝업 효과 ]**
> 🎉 "참 잘했어요!" (정답 폭죽) / 🤔 "다시 한번 생각해 볼까요?" (오답 격려)
>
> **[ 🎬 시각화 리플레이 (Canvas) ]**
> * 애니메이션 처음부터 재현 (예: 3조각 칠해지고, 이어서 2조각 칠해짐)
>
> **[ 📚 단계별 풀이식 (Step-by-Step) ]**
> * Step 1: 처음에 먹은 피자 = 3/8
> * Step 2: 나중에 먹은 피자 = 2/8
> * Step 3: 3/8 + 2/8 = 5/8 (숫자가 합쳐지는 모션 효과)
>
> **[ 하단 영역 : 내비게이션 ]**
> 🔁 [ 다시 풀기 ] (오답 시) | ▶️ [ 다음 문제 ]

### 4. 수학 일기 (Math Diary View - 학습 기록)
> **[ 상단 영역 ]**
> 📅 2026년 3월 학습 현황 | 🔥 5일 연속 열공 중!
> 
> **[ 중앙 영역 : 학습 타임라인 ]**
> * **오늘의 도전 (최근)**
>   - [3단원: 소수의 덧셈] 5문제 완료 (정답률 80%)
> * **어제의 기록**
>   - [2단원: 각도] 10문제 완료 (정답률 100%) 👉 🏆 '각도 마스터' 획득
>
> **[ 하단 영역 ]**
> 📥 [ 지난 풀이 다시보기 ]

### 5. 오답 리플레이 및 리마인드 (Smart Remind View)
> **[ 상단 타이틀 ]**
> 🔍 **비슷한 문제를 만났어요! 기억하고 있나요?**
> 
> **[ 중앙 영역 : 비교 캔버스 ]**
> | 과거의 기록 (오답) | 현재의 도전 |
> | :--- | :--- |
> | 1/4 + 2/4 = **3/8** ❌ | 2/5 + 1/5 = ( ? ) |
> | *(과거 오답 애니메이션 재생)* | *(새로운 애니메이션 대기)* |
> 
> **[ 메시지 영역 ]**
> 💬 "지난번에는 피자 조각(분모)까지 더해서 틀렸었어. 이번에는 조각의 개수(분자)만 합쳐볼까?"
> 
> **[ 하단 버튼 ]**
> 🟠 [ 기억났어! 풀러 가기 ]

## ⚙️ 시스템 및 데이터 관리 로직
* **오답 태깅 시스템:** 모든 문제에 개념 키워드(#받아올림_있음, #분모가_같은_분수 등)를 부여하고, 특정 태그 3회 이상 오답 시 '취약 개념'으로 자동 분류.
* **풀이 데이터 JSON 저장:** 사용자의 오답 과정과 입력 행동을 JSON 형태로 저장하여, 추후 리마인드 화면에서 애니메이션으로 정확히 재현(Replay)할 수 있도록 설계.
