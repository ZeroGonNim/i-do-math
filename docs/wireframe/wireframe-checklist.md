# 와이어프레임 — 경량 검수 체크리스트

> 모든 와이어프레임 작업 완료 후, 다음 항목을 자가 검증한다. 이슈 발견 시 즉시 수정 후 결과를 보고한다.
> ⚠️ **자기평가 편향 주의**: 직접 만든 결과물이므로 "잘 됐을 것"이라는 가정을 버리고, 체크리스트 항목별로 객관적으로 판단한다.

---

## 1. 텍스트 표시 (⛔ 최우선 확인)

> 이 섹션의 모든 항목이 통과되지 않으면 나머지 검수를 진행하지 않는다.

- [ ] **모든 텍스트**에 `resize_node`가 적용되었는가 (세로 1글자 표시 = resize_node 누락)
- [ ] 보드 내 텍스트 너비가 **너비 참조 테이블**(wireframe-rules.md §2)과 일치하는가
- [ ] DESC 패널 내 텍스트 너비가 **392px** (불릿 364px, 서브항목 352px)인가
- [ ] 팝업/모달 내 텍스트 너비가 `팝업w - 패딩×2`로 설정되었는가
- [ ] 텍스트 수정/재작성 시에도 **3단계(create → resize → move)** 전부 수행했는가

### 텍스트 문제 발견 시 수정 절차

```
1. 문제 텍스트 노드 삭제
2. create_text (정렬 파라미터 포함)
3. resize_node (너비 참조 테이블에서 값 확인)  ← ⛔ 절대 생략 금지
4. move_node (y좌표 배치)
5. 수정한 텍스트 아래의 모든 요소 y좌표 cascade 업데이트
```

> ❌ 절대 하지 말 것: 텍스트만 수정하고 resize_node를 생략하는 것
> ❌ 절대 하지 말 것: 한 텍스트만 고치고 아래 요소의 y좌표를 방치하는 것

---

## 2. DESC 패널 항목 배치 (두 번째 우선)

> 텍스트 표시가 통과된 후 이 섹션을 검수한다.

- [ ] DESC 항목이 **위→아래 순서**로 배치되었는가 (겹침 없음)
- [ ] 각 항목의 y좌표가 `이전 항목.y + 이전 항목.height + gap`으로 계산되었는가
- [ ] 같은 그룹 내 간격이 **4px**, 그룹 간 간격이 **16px**인가
- [ ] 2줄 이상 텍스트의 height가 줄 수에 맞게 설정되었는가 (1줄=20, 2줄=36, 3줄=52)
- [ ] 항목 수정 후 **하위 모든 항목**의 y좌표가 cascade 업데이트되었는가

---

## 3. 컴포넌트 내부 정렬 (세 번째 우선)

- [ ] **버튼 텍스트**: resize_node의 width가 **버튼 너비와 동일**한가 (동일해야 CENTER 정렬 작동)
- [ ] **버튼 텍스트**: y좌표가 `(버튼높이 - lineHeight) / 2`로 계산되었는가
- [ ] **헤더 요소**: 아이콘/타이틀 y좌표가 `(헤더높이 - 요소높이) / 2`로 계산되었는가
- [ ] **헤더 타이틀**: resize_node width가 아이콘 영역을 제외한 나머지로 설정되었는가
- [ ] **탭바 텍스트**: resize_node width가 `탭바너비 / 탭 개수`로 설정되었는가
- [ ] **양쪽 정렬 행**: 좌측(LEFT) + 우측(RIGHT)의 resize_node width가 **콘텐츠 전체 너비와 동일**하고 x좌표도 동일한가
- [ ] **금액/날짜 행**: 좌측 라벨과 우측 값의 y좌표가 **동일**한가 (같은 행에 위치)
- [ ] InputField/ListItem 텍스트가 **LEFT + 수직 CENTER**인가
- [ ] 콘텐츠가 패딩 영역을 침범하지 않는가
- [ ] 콘텐츠시작y가 디바이스별로 올바른가 (DT:64, LT:60, TB:80, MB:100)

## 4. 구조 & 완성도
- [ ] 스펙의 모든 `▶` 변형 화면이 누락 없이 생성되었는가 (▶ 없는 화면은 Step B 폴백 적용)
- [ ] 배지 번호와 Description 항목 번호가 1:1 일치하는가 (번호 점프 허용)
- [ ] Description 항목이 기획서 원문 그대로인가 (요약·의역·재작성 금지)
- [ ] 네이밍 규칙(`{ID}-{디바이스}`, `{ID}-{디바이스}-DESC`) 준수하는가

## 5. 좌표 & 배치
- [ ] 신규 보드가 기존 보드와 겹치지 않는가 (충돌 체크)
- [ ] DESC 패널 내 요소가 **로컬좌표**로 배치되었는가
- [ ] Description 항목 간격이 올바른가 (그룹 내 4px, 그룹 간 16px)
